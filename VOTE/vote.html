How it works?

  • Each user enters the election voting portal website.
  • The candidate to be voted for is selected.
  • The user name is inputted in the box provided.
  • User name can't be a number.
  • The candidate is voted for by clicking the vote button.
  • To see current the updated vote number...
  • ...the view current button is clicked
  • The new president is known by clicking the New president button
  • The reset vote button resets all votes permanently
  • Some buttons get disabled when others are clicked
  • The votes are permanently stored even after...
  • ...reloading the page unless resetted.
